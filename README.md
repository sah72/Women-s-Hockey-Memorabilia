# Women-s-Hockey-Memorabilia

**Live Vercel Link:** https://women-s-hockey-memorabilia.vercel.app/

## Overview
A full stack web application that allows users to browse and purchase women’s hockey memorabilia. The application includes dynamic cart functionality, order management, and data storage using a MongoDB database.

---

## Features
- Browse memorabilia items
- Dynamic shopping cart (add/remove/update items)
- Submit and manage orders
- Contact form with database storage
- Responsive design for different screen sizes

---

## Functional Requirements
- Users can view available products
- Users can add items to a shopping cart
- Users can remove or update item quantities in the cart
- Users can submit an order
- Users can search or filter content (if applicable)
- In the track order section you can edit or delete the order
- Contact form submissions are stored in the database
- System validates user input before processing

---

## Technical Stack
**Frontend:**
- HTML
- CSS
- JavaScript

**Backend:**
- Python (Flask)

**Database:**
- MongoDB (MongoEngine)

---

## Key Functionality
- Dynamic DOM manipulation using JavaScript to update cart contents
- RESTful API routes in Flask for handling orders and form submissions
- Integration with MongoDB for persistent data storage
- Error handling and input validation for user actions
